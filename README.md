# 青苗题库

一个可本地运行、也可公开部署的毛概选择题刷题网页。无需登录，公共题库由网站统一提供；每位使用者的进度、错题、收藏和笔记保存在自己的浏览器中。

公共部署说明见 `DEPLOY.md`。

## 启动

双击 `start.bat`，浏览器会打开：

```text
http://127.0.0.1:4173
```

保持命令行窗口开启。关闭窗口后网页服务停止，但学习数据不会丢失。

## 支持的文件

- `.docx`
- `.pdf`
- `.txt`
- `.md`
- `.json`

题库文本推荐使用以下格式：

```text
1. 题干
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
解析：答案依据
知识点：所属章节或概念
```

也支持选项在同一行、`A、`、`（A）`、`答案为 A`、`【答案】A`，以及 Word 表格中分别设置题干、选项 A-D、答案等列的格式。

扫描版 PDF 本质上是图片，浏览器无法直接读取其中的文字，需要先使用 WPS 或其他工具执行 OCR 后再导入。

没有识别到选择题的文件会作为普通复习资料保存，可在“知识点”页面全文检索。

## 数据备份

点击网页右上角“导出备份”，会得到 JSON 文件。之后可通过“导入题库”重新导入该备份。
